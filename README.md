# AI Ticket Scanner 🎟️
A modern, web-based QR code scanner that uses Google's Gemini AI to authenticate event tickets and verify journey dates.

## Features
* Live Camera QR Scanning (using html5-qrcode)
* Static Image Upload support
* Python Flask Backend
* Gemini 2.5 Flash AI integration for messy text parsing
* Responsive UI built with Tailwind CSS
